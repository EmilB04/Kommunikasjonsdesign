The Game of Modes

This webpage demonstrates the storytelling modes from our lectures. Each section both uses the mode and explains how it was implemented. Together, these modes create the toolbox of digital storytelling.

Skrevet av: Emil Berglund

Publisert: 25. september 2025

Text

Example story hook: “The door had just been kicked in. Police outside shouted the word that would define the night: 'Knife!'”

Implementation: Text is the foundation, added simply in <p> tags.

Photo

Implementation: Added with the <img> tag. The photo helps situate the story visually. The image is dynamically fetched from Picsum Photos.

Audio

Implementation: Embedded with <audio>. Audio can add ambience or convey emotion through recorded voices.

Video

Implementation: Inline video via <video>. It provides movement and context for the context for the scene. Source: W3Schools — HTML5 Video.

Map

Implementation: Embedded with a Google Maps iframe. Maps anchor the story in a specific location.

Timeline

  • 19:00 — Michelsen calls ER
  • 20:30 — Fire alarm starts
  • 21:00 — Police arrive
  • 21:12 — Three shots fired

Implementation: Ordered events are presented as an HTML list to show sequence over time.

Data

{
  "year": 2019,
  "police_killings": 6,
  "city": "Bergen"
}

Implementation: Displayed raw data inside a <pre> block to preserve structure and formatting.

Chart

Implementation: Generated via QuickChart API and embedded as an image to visualize numerical data.

Note: The data shown in this chart is fictional and for demonstration purposes only. It does not represent actual statistics.

Infoviz

Implementation: This infographic was generated dynamically with QuickChart.io. Infoviz condenses numbers into a visual form, making it instantly more understandable.
